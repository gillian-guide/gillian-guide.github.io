    Drag-and-Drop Archive#

    The Drag-and-Drop Archive (also known as Gillian's Modpack) is a complete ready-to-play modpack for the best singleplayer-only GTA IV campaign experience. All you have to do to install it is just drop the files into the game folder and do a few extras - thus the name.

    Make sure the prerequisites are met and preparation was done. If you missed one of the steps, you may encounter issues using this archive and won't receive support.

    Installation#

    Which game version to pick?

    The following selector is for the game version - see here for their base differences. The mods in the archives are mostly identical, but 1.0.8.0 includes more Quality of Life changes at the cost of some fixes from FusionFix and removes the Rockstar Games Launcher DRM, while 1.2.0.59 is considered to be more stable and polished.

    My personal preference for a vanilla-faithful playthrough is the 1.2.0.59 archive, and 1.0.8.0 archive for a vanilla+ playthrough (a replay, maybe).

    Last updated: 26.01.2025

    • This archive does NOT provide the best possible performance - the goal is first and foremost to provide the best vanilla-faithful experience. If you just want the best performance - mod the game manually.
    • Do not expect support if attempting to install any more mods that aren't from additional mods - you will likely be denied.
    • Do not attempt to install this version on a downgraded copy.
    • If using Linux, skip steps 3-6 and apply launch options manually.
    1. Download the archive (mirror) (3.63GB, SHA512: fc0689facafe6119fe405895072889b4b1c0f77d2d10a21f0679e22cdf38817a39f6dfa0c12d9d06c6bc9bf114803d2cc3dd4a2e7e6458eb5021c2b47460b653).
    2. Old version in case of stability issues (SHA512: ccf0a3eaa7987b504ec269a835d397b1913131136ad6129735013cb6268fd580d1350b9b37c695846f28cef145f85e03ed67987f528c65d389dae86cc92b923a).
    3. Extract the 1.2 archive.7z archive into the game folder (the one that includes GTAIV.exe).
    4. Launch GTAIVSetupUtilityWPF.exe. It may require you to install .NET 6 Runtime.
    5. Press Open, select the same game folder.
    6. Press Install DXVK and then Setup launch options. Don't touch the toggles unless you are confident in what you are doing.
    7. After pressing Setup launch options and pressing OK you will have launch options in your clipboard, so do one of the following:
      • Steam: Right click the game in your library, press Properties... and paste the contents in the Launch options field.
      • Rockstar Games Launcher: Open the game page in your library, open settings and paste the contents in the Launcher arguments field.
      • Shortcut: Right click on the game shortcut, click Properties and paste the contents at the end of the Target field.
    8. You are ready to play!
      • Launch the game via Steam, Rockstar Games Launcher or the PlayGTAIV.exe executable.
      • It's preferable to start a new savefile. Existing savefiles can work, but you may encounter issues.
      • Set Traffic Density to 100 to take full advantage of the custom popcycle.
      • If you want more mods, check out additional mods.
