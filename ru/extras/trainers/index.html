    Трейнеры#

    Трейнеры - это, по сути, менюшки добавляющие множество интересных и полезных опций. Они также являются лучшей альтернативой очень ограниченному количеству читкодов, предлагаемых игрой. Вам может понадобиться один из них для мультиплеера, но убедитесь, что ваш сервер разрешает их использование.

    ZMenuIV#

    Совместимость

    Этот трейнер не совместим с Complete Edition. Совершите даунгрейд, если вы используете Complete Edition.

    Самый мощный и полноценный трейнер для GTA IV. Также в комплект входит Chaos-мод для стримеров и веселья. И... физика из Goldsrc...? Бхоппинг по фану, ig.

    Установка
    • Установите последний Ultimate ASI Loader.
    • Перейдите на сайт Zolika1351's Zone.
    • Прокрутите страницу до конца и скачайте последнюю версию.
    • Распакуйте ZMenuIV_vx.zip в папку с игрой (исключая папки asi loader if not using xliveless, for developers, old version (has VR support), parachute support(all into game folder). Прочтите readme.txt для деталей на эти папки).
    Аддоны

    Прочтите readme.txt для деталей на аддоны.

    Использование
    • Управление по умолчанию:
      • F7 для переключения меню.
      • Num 8, Num 2, Num 4, Num 6 для навигации. Num 5 - принять, Num 0 - назад.
      • Right Ctrl+Num 1 для ремонта транспорта.
      • Right Ctrl+Num 2 для переворота транспорта.
      • ] (удерживать) для ускорения транспорта.
      • F6 для Airbreak(noclip)
      • Right Ctrl для Special God 1.
      • F1 для Special God 2.
      • 0 для Always God Mode(бессмертие).
      • [ для Never Wanted(откл. розыск).
      • ; для Unlimited Ammo(бесконечные патроны).
      • = для Collision.
      • - для Resurrect(возрождения).
      • ' для Forward Through Door.
      • M для Ragdoll.

    Вы можете добавлять свои собственные привязки клавиш, которые могут сочетать в себе несколько опций. Для этого включите функцию Custom Keybind Creator в разделе Trainer Settings, выделите нужную опцию и нажмите Left Ctrl+Enter для установки привязки.

    Отредактируйте категорию [Keybinds] в ZMenuIV.ini для изменения существующих привязок.

    Привязки клавиш без Numpad

    Не у всех есть Numpad, я понимаю, 80% (и меньше) клавиатуры - это круто и удобно. Откройте ZMenuIV.ini и найдите эти строки:

    [Keybinds]
AirbreakUp=87
AirbreakDn=83
AirbreakForward=104
AirbreakBack=98
AirbreakLeft=100
AirbreakRight=102
MenuUp=104
MenuDn=98
MenuLeft=100
MenuRight=102
MenuEnter=101
MenuBack=96

    и измените их на:

    [Keybinds]
AirbreakUp=32
AirbreakDn=160
AirbreakForward=87
AirbreakBack=83
AirbreakLeft=65
AirbreakRight=68
MenuUp=73
MenuDn=75
MenuLeft=74
MenuRight=76
MenuEnter=13
MenuBack=220
    • Что будет:
      • IJKL для навигации. Enter - принять, \ - назад.
      • WASD для горизонтального управления Airbreak, Space для поднятия высоты, и Left Shift для спуска.

    Liberty's Legacy#

    Совместимость

    Данный тренер совместим с Complete Edition, а также с патчами 1.0.8.0 и 1.0.7.0.

    Этот трейнер не такой мощный, как ZMenuIV, но все же предлагает много полезных функций, особенно если вы играете с Complete Edition.

    Установка
    • Установить последние Ultimate ASI Loader и ScriptHook.
    • Перейдите на страницу GTAForums.
    • Скачайте последнюю версию.
    • Распакуйте Liberty's Legacy Trainer xxx.zip в папку с игрой.
    Использование
    • Управление по умолчанию:
      • F11 или RB+X/R1+Квадрат для переключения меню.
      • Стрелки или крестовина для навигации. Enter или A/X - принять, Backspace или B/Кружок - назад.

    Дополнительные привязки клавиш можно установить, выделив нужную опцию, нажав Left Ctrl+Enter и нажав клавишу, которую вы хотите привязать.

    Simple Native Trainer#

    Совместимость

    Данный трейнер совместим со всеми патчами.

    Этот трейнер менее полный, чем два других, но... некоторым людям он будет полезен, я думаю.

    Установите
    • Установить последние Ultimate ASI Loader и ScriptHook.
    • Перейдите на страницу GTAForums.
    • Скачайте последнюю версию (где-то в посте).
    • Распакуйте trainer.asi, trainer.ini, trainertbogt.asi, trainertbogt.ini, trainertlad.asi, trainertlad.ini из trainervxx.rar в папку с игрой.
    Использование

    F3 для открытия меню, Num 8, Num 2, Num 4, Num 6 для навигации. Читайте readme.doc или страницу GTAForums для инструкций для изменения клавиш.
